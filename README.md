# AI 每日简报 (Daily AI News)

每天自动生成的 AI/科技领域新闻简报，由 QoderWork 定时任务驱动。

## 浏览方式

- **最新简报**: [https://polors.github.io/daily-ai-news/](https://polors.github.io/daily-ai-news/)
- **历史归档**: [archive/](./archive/) 目录下按日期存放

## 内容板块

- 🌊 AI 动态与趋势
- 📰 AI 今日看点
- 🔥 AI 大事件
- 🛠️ AI 应用前线
- 📊 数据速递
- 🔬 ArXiv 今日精选论文
- 🚀 GitHub AI 趋势日榜
- 💡 今日洞察

## 技术栈

- 数据来源：WebSearch + Hacker News + ArXiv + GitHub Trending + HuggingFace
- 生成方式：QoderWork 定时任务 (每日 8:40 CST)
- 托管：GitHub Pages
